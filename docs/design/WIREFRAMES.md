## RESPONSIVE CONSIDERATIONS (PWA)

### MOBILE (< 640px):
- Single column layout
- Full-width cards
- Bottom navigation (if needed later)
- Font size: 16px min (iOS legibility)

### TABLET (640px - 1024px):
- Same layout but more padding
- Cards max-width: 600px, centered
- Larger image previews

### DESKTOP (> 1024px):
- Max content width: 480px
- Centered layout (mobile-first)
- "Best viewed on mobile" hint

- Min 44x44px (iOS guideline)

- Text on backgrounds: 4.5:1 minimum
- Primary buttons: High contrast

- All images have alt text
